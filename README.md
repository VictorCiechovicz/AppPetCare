# ****💻 AppPetCare****

 App to control of animal for adoption. This application was created for the purpose of studies.
 
## ****✨ Stack****

- React Native
- Typescript
- Expo Bare Workflow
- React Native SVG
- Phosphor Icons
- React Native ImagePicker
- Expo Google Fonts
- React Navigation
- Firebase Cloud Firestore
- Firebase Cloud Auth
- Firebase Storage

## **🛠️ Features**

- [ ] User authentication (customer and waiter)
- [ ] User Register
- [ ] Password Recovery with e-mail
- [ ] Pets Register and Update info
- [ ] Pets image upload
- [ ] User perfil
- [ ] List of Pets adotacion
- [ ] Manage Orders open and close
- [ ] Screen Routes
- [ ] And more...

## 🔧 ****Running****

Clone this project and start with Expo:

```jsx
  git clone
  npm install
  expo run:androind
```
Start App with run-android
```jsx
 npx react-native run-android
```
